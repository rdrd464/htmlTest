HTML course

HTML is a way of writing and structuring web pages, and HTML is not a programming language in the actual sense, but rather a descriptive language, that is, it is used to describe the page and distribute and arrange its content of text, links, images, videos, etc., and it is the main component of any website. HTML uses the so-called tags or Tags, which are codes, each with a special task that it performs, some of which display the title, some display texts, and some display images, links, videos, and so on, and the browser understands these codes or tags and displays them in the specified and required form.

Course objectives

  • Understand the concept of HTML
  • Learn each tag and its purpose
  • Learn how to add text, videos and other usual content in web pages

Course Requirements

  • Browser (Safari and Google Chrome)
  • text editor
